Values Defines the height as an absolute value. Defines the height as a percentage of the containing block's height. auto The browser will calculate and select a height for the specified element. max-content The intrinsic preferred height. min-content The intrinsic minimum height. fit-content() Uses the fit-content formula with the available space replaced by the specified argument, i.e. min(max-content, max(min-content, )). Accessibility concerns Ensure that elements set with a height aren't truncated and/or don't obscure other content when the page is zoomed to increase text size.
