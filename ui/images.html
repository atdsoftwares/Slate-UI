Images

Images can be responsive to fit the parent's width, and also can be customised to be round shaped

Example of Responsive Images

You can add class responsive-image to make your image fit the width of container. It's height will get adjusted by keeping the aspect ratio same. If you want to change the aspect ratio, you will have to crop the image
